# Lumière - Luxury Fashion Brand Website

A minimalist, elegant e-commerce website for a luxury clothing brand featuring women's, men's, and children's collections. This website embodies old money aesthetic with a focus on refined simplicity and timeless elegance.

## 🏛️ Design Philosophy

**Lumière** represents luxury through restraint:
- **Minimalist Design**: Clean lines, ample whitespace, and intentional simplicity
- **Old Money Aesthetic**: Understated elegance, refined typography, and classic color palette
- **Timeless Focus**: Investment pieces over trends, quality over quantity
- **Sophisticated Appeal**: Designed for discerning customers who appreciate true luxury

## 🎨 Design Features

### Color Palette
- **Cream** (#f5f1ed) - Warm, elegant base
- **White** (#ffffff) - Clean, premium space
- **Black** (#1a1a1a) - Timeless sophistication
- **Gold** (#b8860b) - Subtle luxury accents
- **Light Gray** (#e8e4e0) - Soft borders and dividers

### Typography
- **Serif Font** (Garamond): Headlines, brand name
- **Sans-serif Font** (Segoe UI): Body text
- Generous letter-spacing and line-height throughout

## 🚀 Getting Started

Open `index.html` in your browser. No installation required!

Or use a local server:
```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## 🛠️ Customization

Update:
- Brand name (search and replace "LUMIÈRE")
- Colors in `css/styles.css`
- Product information in each collection page
- Contact details in `pages/contact.html`
